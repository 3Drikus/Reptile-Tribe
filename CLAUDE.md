# Reptile Tribe - Project Brief

## Business
Reptile Tribe is a premium reptile breeder and seller based in Pretoria, South Africa.
They specialise in Geckos, Chameleons, Iguanas and other exotic reptiles.
Target audience: passionate reptile enthusiasts and collectors in South Africa.

## Brand Direction
Think: Apple meets African wilderness. Brutally simple and modern, but with warmth, soul and a tribal community feel.
NOT a generic pet store. This is a premium, curated exotic reptile experience.
Tone: trustworthy, passionate, knowledgeable. Like a tribe of collectors, not a shop.

## Colours
Background: #0f1a0e (deep forest black-green)
Surface: #162114 (dark green)
Border: #2a3d28 (muted green)
Accent: #c17f3a (warm terracotta/amber)
Accent Light: #e8a85a (golden amber)
Text: #f0ebe0 (warm cream)
Text Muted: #8a9e82 (sage green)

## Typography
Heading Font: Fraunces (Google Fonts) - for display headings
Body Font: DM Sans (Google Fonts) - for all body text
Use next/font/google for both fonts

## Stack
- Next.js 14 App Router
- TypeScript
- Tailwind CSS with custom theme
- shadcn/ui components

## Design Rules
- Mobile-first always
- Generous white space - let things breathe like Apple
- No Lorem Ipsum - use real reptile business copy
- Every section must be a separate component in /components
- Use semantic HTML for SEO (h1, h2, section, article, nav, footer)
- Every image must have descriptive alt text
- Subtle animations only - nothing flashy
- Rounded corners: 12px consistently
- One H1 per page only
- Always use Lucide React for icons, import from 'lucide-react'. Never use emojis as icons.
- Icon size default: 24px
- Icon colour: use terracotta #c17f3a for accent icons, sage #8a9e82 for subtle/secondary icons
